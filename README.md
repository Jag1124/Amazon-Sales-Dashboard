# Amazon Sales Analysis Dashboard 📊
## 📌 Project Overview
This project focuses on analyzing Amazon sales data using Excel, SQL, and Power BI. The goal was to extract meaningful insights and visualize them through an interactive dashboard.
## 🛠️ Tools & Technologies Used
- Microsoft Excel (Pivot Tables & Charts)
- SQL (PostgreSQL)
- Power BI
- Google Colab (Python for analysis)
## 📊 Key Features
- Category-wise sales analysis
- Order status distribution (Shipped, Cancelled, Pending)
- Sales trend over time (monthly analysis)
- Data cleaning and transformation
## 📈 Insights Gained
- Identified top-performing product categories
- Analyzed order status distribution
- Observed monthly sales trends
- Improved understanding of data visualization
## 🚀 Conclusion
This project helped in developing strong skills in data analysis, visualization, and handling real-world datasets.
